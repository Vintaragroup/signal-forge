# KPI Tracking

## Core Metrics

| Metric | Target | Notes |
| --- | ---: | --- |
| Opportunities sourced |  | sponsors, guests, partners |
| Reviewed opportunities |  | operator review |
| Pursued opportunities |  | approved outreach |
| Pitches sent |  | manual send |
| Replies |  | positive or neutral |
| Follow-ups needed |  | next touch required |
| Booked calls |  | sponsor or partner calls |
| Closed won |  | deal, placement, collaboration |
| Closed lost |  | declined or poor fit |

## Quality Metrics

- Average opportunity score
- Audience fit rate
- Sponsor category fit
- Reply rate by opportunity type
