# KPI Tracking

## Core Metrics

| Metric | Target | Notes |
| --- | ---: | --- |
| Leads imported |  | source volume |
| Leads reviewed |  | human review completion |
| Pursued leads |  | approved for outreach |
| Research more |  | data quality gap |
| Sent outreach |  | manually sent |
| Replies |  | positive or neutral |
| Booked calls |  | advisory conversations |
| Closed won |  | policy or client win |
| Closed lost |  | not a fit or no sale |

## Quality Metrics

- Average lead score
- Percentage with risk signal
- Percentage with website
- Percentage with clear recommended offer
