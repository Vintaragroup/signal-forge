# KPI Tracking

## Core Metrics

| Metric | Target | Notes |
| --- | ---: | --- |
| Opportunities sourced |  | venues, curators, media, partners |
| Reviewed opportunities |  | human review completed |
| Pursued opportunities |  | approved outreach |
| Pitches sent |  | manual send only |
| Replies |  | positive or neutral |
| Follow-ups needed |  | no reply or next step |
| Booked calls |  | partner or booking calls |
| Confirmed wins |  | placement, show, partner |
| Closed lost |  | declined or poor fit |

## Quality Metrics

- Average opportunity score
- Genre fit rate
- Contact confidence rate
- Reply rate by persona

Use the current pipeline report until an artist-specific report is scoped.
